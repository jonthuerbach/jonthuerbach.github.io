CSS

Global CSS settings and fundamental HTML elements

Overview / Getting Started

Make sure the following elements are in your project's code to ensure the best experience by users.

HTML5 doctype

UA Bootstrap makes use of certain HTML elements and CSS properties that require the use of the HTML5 doctype. Include it at the beginning of all your projects.

<!DOCTYPE html>
<html lang="en">
  ...
</html>

Mobile First

To ensure proper rendering and touch zooming, add the viewport meta tag to your <head>

<meta name="viewport" content="width=device-width, initial-scale=1">

Typography

Headings

All HTML headings, <h1> through <h6>, are available. .h1 through .h6 classes are also available, for when you want to match the font styling of a heading but still want your text to be displayed inline.

Example

h1. UA Bootstrap Heading 1

h2. UA Bootstrap Heading 2

h3. UA Bootstrap Heading 3

h4. UA Bootstrap Heading 4

h5. UA Bootstrap Heading 5
h6. UA Bootstrap Heading 6
<h1>h1. UA Bootstrap Heading 1</h1>
<h2>h2. UA Bootstrap Heading 2</h2>
<h3>h3. UA Bootstrap Heading 3</h3>
<h4>h4. UA Bootstrap Heading 4</h4>
<h5>h5. UA Bootstrap Heading 5</h5>
<h6>h6. UA Bootstrap Heading 6</h6>

Body Copy

UA Bootstrap's global default font-size is 16px, with a line-height of 1.5. This is applied to the <body> and all paragraphs. In addition, <p> (paragraphs) receive a bottom margin of 16px by default.

Example

Nullam quis risus eget urna mollis ornare vel eu leo. Cum sociis natoque penatibus et magnis dis parturient montes, nascetur ridiculus mus. Nullam id dolor id nibh ultricies vehicula.

Buttons

Options

Use the following button classes to quickly create a styled button.

Example

<!-- Red button -->
<button type="button" class="btn btn-red">Red</button>

<!-- Blue button -->
<button type="button" class="btn btn-blue">Blue</button>

<!-- Cactus button -->
<button type="button" class="btn btn-cactus">Cactus</button>

<!-- Sky button -->
<button type="button" class="btn btn-sky">Sky</button>

<!-- River button -->
<button type="button" class="btn btn-river">River</button>

<!-- Sand button -->
<button type="button" class="btn btn-sand">Sand</button>

<!-- Mesa button -->
<button type="button" class="btn btn-mesa">Mesa</button>

<!-- Brick button -->
<button type="button" class="btn btn-brick">Brick</button>

Sizes

The button sizes are large, default, small, and extra small.

Example

<!-- Large buttons -->
<button type="button" class="btn btn-red btn-lg">Large Button</button>
<button type="button" class="btn btn-blue btn-lg">Large Button</button>

<!-- Default buttons -->
<button type="button" class="btn btn-red">Default Button</button>
<button type="button" class="btn btn-blue">Default Button</button>

<!-- Small buttons -->
<button type="button" class="btn btn-red btn-sm">Small Button</button>
<button type="button" class="btn btn-blue btn-sm">Small Button</button>

<!-- Extra Small buttons -->
<button type="button" class="btn btn-red btn-xs">Extra Small Button</button>
<button type="button" class="btn btn-blue btn-xs">Extra Small Button</button>

Anchor Buttons

You can use the same classes for anchor buttons.

Example

Large Button Large Button

Default Button Default Button

Small Button Small Button

Extra Small Button Extra Small Button 

<!-- Large buttons -->
<a href="#" class="btn btn-red btn-lg">Large Button</a>
<a href="#" class="btn btn-blue btn-lg">Large Button</a>

<!-- Default buttons -->
<a href="#" class="btn btn-red">Default Button</a>
<a href="#" class="btn btn-blue">Default Button</a>

<!-- Small buttons -->
<a href="#" class="btn btn-red btn-sm">Small Button</a>
<a href="#" class="btn btn-blue btn-sm">Small Button</a>

<!-- Extra Small buttons -->
<a href="#" class="btn btn-red btn-xs">Extra Small Button</a>
<a href="#" class="btn btn-blue btn-xs">Extra Small Button</a>
